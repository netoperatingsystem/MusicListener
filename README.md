# MusicListener

Application for listening music

## Features: 

- Playing songs
- Support for \*.mp3 and \*.wav formats
- Listing all songs from device
- Displaying current and total song time
- Repeat song
- Deleting songs
- Shuffle song list
- Albums tab
- Search bar
- Album playlist
